**Primary Users:** Users who need to ask questions about data in Excel files  
**Tech Constraints:** Backend: Python FastAPI. Frontend: Next.js. Database: PostgreSQL with pgvector extension. Deploy: Docker Compose (backend + frontend + postgres services). Use text-embedding-3-small (OpenAI) for embeddings and gemini-2.0-flash (Google Generative AI SDK) for answer generation.

Project Summary: This project involves building a web application that allows users to upload Excel files and ask natural language questions about the data within them. The system processes the files by splitting the content into text chunks, converting them to vectors, and storing them for retrieval. When a user asks a question, the system finds relevant data chunks, generates an answer using an AI model, and presents the response along with citations showing the source file and data location.
